# DWBB(DesignWare Building Block)

[The DesignWare Library's Datapath and Building Block IP](https://www.synopsys.com/dw/buildingblock.php) is a collection of reusable intellectual property blocks that are tightly integrated into the Synopsys synthesis environment. This is its wrapper in Chisel.
